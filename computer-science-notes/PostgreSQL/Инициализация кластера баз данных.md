При инициализации кластера баз данных([[Кластер баз данных]]) в [[PGDATA]] создаются три одинаковые базы данных:
- [[template0]] используется, например, для восстановления из логической резервной копии ([[Логическая резервная копия]]) или для создания базы в другой кодировке и никогда не должна меняться;
- [[template1]] служит шаблоном для всех остальных баз данных, которые может создать пользователь в этом кластере;
- [[postgres]] представляет собой обычную базу данных, которую можно использовать по своему усмотрению.
